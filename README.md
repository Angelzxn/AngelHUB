# Angel HUB 🔥

Premium automation hub for **World Fighters** (Roblox).

## Quick Start

Paste this in your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/AngelDev-EU/AngelHUB/main/AngelHUB.lua"))()
```

## Features

| Module | Features |
|--------|----------|
| **Main** | Auto Click, Auto Collect Drops, Auto Farm (nearest mob) |
| **Gamemodes** | World Teleport, Trial Join, Auto Trial Loop |
| **Gacha** | Auto Gacha Roll (Haki), Auto Stars/Pets |
| **Quests** | Position Tracker, Copy Position |
| **Rewards** | Auto Time Rewards, Achievement Claim, Follow Rewards |
| **Misc** | WalkSpeed, JumpPower, Anti-AFK, Player Stats, Hub Info |

## Controls

- **RightControl** — Toggle hub visibility
- **Drag title bar** — Move the hub
- **X** — Destroy hub
- **-** — Minimize hub

## Compatible Executors

Madium, Synapse, KRNL, Fluxus, Wave
